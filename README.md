# 🎮 Free Games Generator

Um site minimalista desenvolvido para testar lógica de programação e ajudar usuários a descobrirem jogos públicos e gratuitos de forma aleatória.

## 🚀 Tecnologias Utilizadas
* HTML5
* CSS3
* JavaScript (ES6+)

## 💻 Como Executar o Projeto
1. Clone o repositório: `git clone https://github.com/GAGABRIELTRUE/Free-Games.git`
2. Abra o arquivo `index.html` no seu navegador.

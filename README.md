# 🎮 Free Games

Um site minimalista desenvolvido para ajudar usuários a descobrirem jogos públicos e gratuitos de forma totalmente aleatória e descompromissada.

## 🔗 Acesse o Site
👉 [Clique aqui para acessar o Free Games](https://github.io)

## 🎯 Sobre o Projeto
Este projeto nasceu como um teste prático de desenvolvimento web focado em dispositivos móveis. O objetivo principal foi criar uma aplicação leve, rápida e direta que resolve o problema de "não saber o que jogar", sorteando links de jogos gratuitos com apenas um clique.

## 🚀 Tecnologias Utilizadas
* **HTML5** - Estruturação das páginas.
* **Tailwind CSS (via CDN)** - Estilização moderna e responsiva focada em agilidade de desenvolvimento mobile.
* **JavaScript** - Lógica de sorteio aleatório dos jogos e manipulação do DOM.

## 📂 Estrutura do Repositório
* `index.html`: Página principal e interface do usuário.
* `scripts/`: Pasta dedicada aos arquivos de lógica em JavaScript.
* `README.md`: Documentação do projeto.

## 💻 Para Desenvolvedores (Opcional)
Se você quiser baixar o código para estudar ou fazer modificações locais:
1. Clone este repositório: `git clone https://github.com`
2. Abra o arquivo `index.html` diretamente em seu navegador (não requer instalação de `node_modules`).
